# 车牌颜色识别原子组件 ## 接口说明 车牌颜色识别原子组件，基于深度学习与图像识别等技术，对一个车牌小图像，识别车牌的颜色类型。功能包括： * 车牌颜色识别 ## 依赖 * anconda == 3.6 * tensorflow-gpu >= 1.9 * opencv_python >= 3.4 ## 使用说明 * 接口方式：HTTP接口,post请求 * url：http://xx.xx.xx.xx:xx/APIService/lprColorService * 接口输入参数：json数据： {"imageBase64":"xx"} | 参数 | 类型 | 说明 | | -------- | -----: | :----: | |imageBase64 | string | 待分析图像的base64编码，针对一个车牌小图像，图像清晰度较高，数据不大于2M | * 接口输出参数： | 参数 | 类型 | 说明 | | -------- | -----: | :----: | |statusCode| string | 状态码 | |log_id| string | 状态id | |timeSecond| int | 分析用时（秒） | |result| array | 车牌信息列表 | |+thingType| string | 车牌的颜色 | |+confidence| float | 置信度 | * 状态码说明： | statusCode值 | 含义型 | | -------- | :----: | | success| 成功| | postData_formError| Post数据格式错误| | imageBase64_formError| Base64编码格式错误| | imageBase64_empty| Base64数据为空| | imageBase64_greaterThan2M| Base64数据大于2M| | overtime| 图像分析超时| | recog_runError| 图像分析运行失败| ## 输入 ![Bootswatch](../demoImages/lprColor/3.jpg) ## 输出示例 {"statusCode": "success", "result": [{"thingType": "单黄牌", "confidence": 0.9999947547912598}, {"thingType": "单绿牌", "confidence": 4.661291768570663e-06}], "log_id": "2019723928_aa738858-acec-11e9-9d10-c3741d03f157"}
